Problema y analisis

Disponemos de un cliente que nos ha pedido una aplicacion de Windows para poder gestionar sitios de comida y poder tener un listado con precios y puntaciones ademas de informacion generica de las ubicaciones de estos.

Tras recibir esta informacion del cliente disponemos a evaluar que datos son necesarios para poder crear la base de la aplicacion.

Primero empezamos a definir que tipo de proyecto vamos a trabajar, para esta ocasion se ha decidido optar por WinUI 3, que es la nueva intefaz grafica que ofrece Microsoft.
