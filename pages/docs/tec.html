Documentacion tecnica

  • Sitio.cs

La clase principal donde marcamos los datos que vamos a trabajar en la estructuracion de los datos.

En esta clase se ha definido un metodo de auto-incremento a una Id para poder realizar posteriormente una logica de trabajo mas comoda que la intervencion de obtener el index de la tabla, ya que si la tabla es ordenada de forma diferente e intentamos realizar un eliminado de datos o modificacion, podriamos obtener otra entrada que no es la que nos interesa.

  • Logica.cs

La clase donde vamos a trabajar la logica de datos con la que mostraremos la lista de objetos.

Esta actualmente para la demostracion disponemos los datos en memoria mediante un ObservableCollection el cual nos permite listar los objetos que se indiquen, ademas de una lista de objetos adjuntada a mano.
*todos los datos que estan en el proyecto son datos de lugares reales, junto a una valoracion personal y aproximada

  • MainPage.xaml.cs

La logica que hemos creado anteriormente debemos mostrarla de alguna forma, una de las clases mas importantes es la MainPage, en esta trabajamos la adicion de los datos generados dentro de la clase Logica.cs para aderirla a la DataGrid para que nos muestre la tabla.

Tambien tenemos definido los metodos para la ordenacion de la tabla por columnas y las acciones para derivar la informacion de los botones Editar y Eliminar.
